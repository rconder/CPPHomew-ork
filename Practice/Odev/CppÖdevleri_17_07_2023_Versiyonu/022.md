#### Aşağıdaki programın çıktısı ne olur? Açıklayınız:
----

- Fonksiyona yapılan her çağırıda bu ifade ayrıca ele alınacak ve statik yerel değişken olan ifadenin bir ifades elde edilecek ve 
- Ekran çıkıtısı: **123** olmalı

```cpp
#include <iostream>

int foo()
{
	static int x{};
	
	return ++x;
}

void func(int i = foo())
{
	std::cout << i;
}

int main()
{
	func();
	func();
	func();
}
```
